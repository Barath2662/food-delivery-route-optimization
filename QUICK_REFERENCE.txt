ALGORITHM:
  • Dijkstra's Shortest Path
  • Time Complexity: O((V+E)log V)
  • Space Complexity: O(V)
  • Optimal Path Guaranteed

GRAPH:
  • 4 Stages (Restaurant → Hub → Zone → Customer)
  • 9 Nodes (sample graph)
  • 10 Edges (sample graph)
  • Forward progression enforced

API (7 Endpoints):
  1. POST /api/optimize          Find optimal route
  2. GET /api/nodes              Get all nodes
  3. GET /api/nodes/<id>         Get node details
  4. GET /api/routes             Get available routes
  5. POST /api/route-details     Get edge details
  6. GET /api/stats              Graph statistics
  7. GET /api/health             Health check
