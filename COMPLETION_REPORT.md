### ✅ REST API (7 Endpoints)
1. `POST /api/optimize` - Find optimal route
2. `GET /api/nodes` - Get all nodes
3. `GET /api/nodes/<id>` - Get node details
4. `GET /api/routes` - Get available routes
5. `POST /api/route-details` - Get edge information
6. `GET /api/stats` - Graph statistics
7. `GET /api/health` - Health check
